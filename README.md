**COS2614 Assignment 3 – Advanced Object-Oriented Programming**

**Overview**

This repository contains my solutions for **COS2614 Assignment 3** completed during my **Bachelor of Science in Computer Science** studies at the **University of South Africa (UNISA)**.

The assessment focuses on implementing advanced object-oriented programming concepts using **C++** and the **Qt Framework** through three software applications. The projects demonstrate inheritance, polymorphism, abstract classes, GUI development, dynamic memory allocation, collections, and software design principles.

- C++

# How to Run

## Requirements

- Qt Creator
- Qt 5 or later
- C++ Compiler (MinGW or MSVC)

---

### Question 1

1. Open **Question1-Payroll-System** in Qt Creator.
2. Build the project.
3. Run the console application.

---

### Question 2

1. Open **Question2-Gate-Access-Control** in Qt Creator.
2. Build the project.
3. Run the GUI application.
4. Enter the password using the keypad.

---

### Question 3

1. Open **Question3-Customer-Management-System** in Qt Creator.
2. Build the project.
3. Run the console application.
4. Search customers by ID, name, or postal code.
